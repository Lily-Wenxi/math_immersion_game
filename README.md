# Math Immersion Game（七年级原型）

这是一个面向中学到高中阶段的数学闯关 Web 应用原型，当前先实现七年级内容。

## 特色

- 基于新加坡数学思路（CPA：具体-形象-抽象）设计题目提示。
- 5 个闯关关卡，必须先通过当前关卡才能解锁下一关。
- 每关奖励积分，并将积分与进度保存在浏览器本地（localStorage）。
- 适合后续按年级扩展：增加八年级、九年级等题库和地图。

## 本地运行

```bash
python3 -m http.server 8000
```

然后打开：`http://localhost:8000`
